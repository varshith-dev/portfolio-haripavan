# portfolio-haripavan

Personal portfolio site for **Hari Pavan** — People & Culture Partner · HR Domain Expert.

Built with [Next.js](https://nextjs.org) (App Router) and Tailwind CSS. A single
long-form page with hero, experience, recognition, an auto-playing highlights
carousel, expertise, education, and contact.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm run start
```

## Deploy

Deploys to [Vercel](https://vercel.com) with zero configuration — import the
repository and it builds automatically. All content lives in
`src/lib/data.ts`; images are in `public/`.
